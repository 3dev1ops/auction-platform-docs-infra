# 서비스 구조도 (IA: Information Architecture)

## 1. 개요
본 문서는 사용자 화면(Presentation Layer)의 메뉴 구조와 진입 경로를 정의합니다.

## 2. 화면 및 메뉴 구성

### 1️⃣ 사용자 계정 및 마이페이지 (User)
*   **로그인 (Login)**
    *   이메일 로그인
    *   소셜 로그인 (구글, 네이버, 카카오)
    *   아이디/비밀번호 찾기
*   **회원가입 (Sign Up)**
    *   약관 동의
    *   정보 입력
*   **마이페이지 (My Page)**
    *   **대시보드**: 신뢰도 지표(매너온도), 보유 예치금 요약
    *   **거래 내역**:
        *   구매 내역 (입찰 중, 낙찰, 구매 확정, 패찰)
        *   판매 내역 (판매 중, 판매 완료, 유찰)
    *   **관심 목록 (Watchlist)**: 찜한 경매 물품
    *   **최근 본 경매**: 최근 조회 서비스 기록
    *   **내 정보 수정**: 프로필 사진, 닉네임 변경

### 2️⃣ 경매 플랫폼 (Auction Platform)
*   **메인 홈 (Home)**
    *   자동 추천 물품 리스트
    *   마감 임박 경매 리스트
*   **물건 찾기 (Search & Browse)**
    *   카테고리별 탐색
    *   검색 (키워드)
    *   필터 (가격대, 지역, 상태 등)
*   **물품 설명 상세 (Detail View)**
    *   상품 이미지/정보
    *   판매자 신뢰도 정보
    *   실시간 입찰 현황 및 입찰 버튼
    *   문의 댓글 영역
